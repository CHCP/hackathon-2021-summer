# 赛凡链项目简介

## 项目介绍

凡链以Substrate为基础框架，提供一个去中心化、开放包容、安全智能的科幻创作应用链，为创作者和产业从业者提供存证、共识、驱动引擎和交易、激励、治理模式，创生并推动独具创意、面向未来的科幻创意产业繁荣发展。
