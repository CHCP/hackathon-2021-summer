# UdunNetwork项目简介

UdunPay Network遵循了“CypherPunk”和“Substrate"架构精神，是新一代分布式互联支付网络。它一方面链接各个主链上的数据，一手链接多方用户，为它们进行互联和数据交互。 在未来，从生态治理，业务重构，应用分发到社区投票，都会有相应的场景应用。
